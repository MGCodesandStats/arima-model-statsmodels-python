# Weather Modelling: Time Series Analysis

This project the use of time series models in forecasting temperature and rainfall patterns.

Data is sourced from [Met Ã‰ireann](https://www.met.ie/climate/available-data/historical-data) and the [UK Met Office](https://www.metoffice.gov.uk/pub/data/weather/uk/climate/stationdata/braemardata.txt).

Techniques used:

- ARIMA and Prophet modeling for temperature forecasting
- LSTM modeling for rainfall forecasting
- Monte Carlo simulations for extreme weather event modeling
